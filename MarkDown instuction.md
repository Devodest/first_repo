# Инструкция для работы с Markdown

## Выделение текста
 
 Что бы текст выделить курсивом, необходимо его обрамить звёздочками (*). Например, *вот так.*

 Что бы выделить текст полужирным необходимо обрамить его двойными звёздочками (**)
 Например, **вот так.**

## Список
Чтобы добавить ненумерованные списки необходимо пункты выделить (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо просто пронумеровать, 
Например, вот так:
1. Первый пункт
2. Второй пункт
